# ComputerCraft-Website

## Project setup

### Goes to the right folder and installs dependencies
```
cd website
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
